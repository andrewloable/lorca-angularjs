# lorca-angularjs
Go Desktop GUI using lorca and AngularJS

## Requirements
Go
Google Chrome

## Build
```
git clone git@github.com:andrewloable/lorca-angularjs.git
cd lorca-angularjs
go get -u
```

###Windows
```
build-windows.bat
```
###MacOS
```
./build-macos.sh
```
###Linux 
```
./build-linux.sh
```
